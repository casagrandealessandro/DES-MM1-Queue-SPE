## DES M/M/1 Queue Simulation

This repository contains a Discrete Event Simulation of an M/M/1 queue, developed for the Simulation and Performance Evaluation course at the University of Trento.

### Requirements
Install dependencies:

```zsh
python3 -m pip install -r requirements.txt
```

### Usage
Run the simulation with:

```zsh
python3 main.py
```

For more details, results, and analysis, please refer to the `Report.pdf` document.
